О боте

Gidrolat — умный бот для управления доступом в закрытый Telegram-чат.
Бот полностью автоматизирует процесс модерации и оплаты подписок

Что умеет бот:

  • Принимать и проверять анкеты пользователей
  • Распределять роли (пользователь, модератор, админ)
  • Принимать оплату и управлять подписками
  • Автоматически выдавать ссылки на чат
  • Показывать статистику

Как работает бот

1 Пользователь нажимает /start
2 Появляется меню бота
3 Пользователь заполняет анкету
4 Анкета отправляется на проверку
5 Модератор проверяет анкету
6 Пользователь выбирает тариф
7 Пользователь оплачивает подписку
8 Модератор подтверждает оплату
9 Бот отправляет ссылку в чат

Модератор

Как стать модератором

1. Отправьте боту команду /id
2. Отправьте полученный ID супер-админу
3. Он добавит вас в список администраторов

Команда для панели

/admin

FAQ

Нет команды /admin
Проверьте, добавлен ли пользователь в список администраторов. Только модераторы имеют доступ к этой команде.
Промокод не работает
Возможные причины:
• Промокод уже использован
• Срок действия истёк
• Промокод не активен
Пользователь был удалён из чата
Это автоматическое действие бота. Пользователь удаляется после окончания срока подписки. Для восстановления доступа необходимо продлить подписку.
Как получить ID пользователя?
Отправьте боту команду /id — бот пришлёт ваш уникальный идентификатор.
