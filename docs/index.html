Обзор

Telegram-бот Gidrolat предназначен для приема анкет, модерации, управления тарифами и подтверждения оплат.

Быстрый старт

Пользователь

/start → заполнить анкету → выбрать тариф

Администратор

/admin → модерация анкет → подтверждение оплат

Супер-админ

Настройки бота → тарифы → промокоды

ID чата

Написать !id в группе

Роли

  • Пользователь — анкета и тарифы
  • Администратор — проверка анкет
  • Супер-админ — управление ботом

Пользователь

  • Анкета — заполнение
  • Статус — статус профиля
  • Тарифы — выбор тарифа

Администратор

  • /admin — открыть админку
  • Модерация анкет
  • Подтверждение оплат

Супер-администратор

  • Редактирование сообщений
  • Тарифы
  • Промокоды
  • Список админов
