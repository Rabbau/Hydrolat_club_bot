# 📋 Реализация системы анкет, платежей и администраторов

## 🎯 Обзор реализованного функционала

Полностью реализована система управления членством в закрытом дискуссионном клубе по гидролатам с поддержкой:
- ✅ Анкетирования пользователей
- ✅ Модерации анкет администраторами
- ✅ Системы платежей и подписок
- ✅ Промокодов (групповых и личных)
- ✅ Управления администраторами (админ и супер-админ)
- ✅ Управления сообщениями бота
- ✅ Управления тарифными планами

---

## 📊 Архитектура БД

### Новые таблицы:

1. **survey_submissions** - Анкеты пользователей
   - Статусы: pending_review → approved → pending_payment → paid
   - Также поддерживает rejected статус
   - Хранит ответы в JSON
   - Поле personal_discount для персональных скидок

2. **payment_plans** - Тарифные планы
   - Ежемесячно, полугодие, год и т.д.
   - Хранит длительность в днях и цену

3. **subscriptions** - Активные подписки пользователей
   - Связь user → plan
   - Отслеживание начала и конца подписки
   - Связь с промокодом (если использован)

4. **promo_codes** - Промокоды
   - Поддержка групповых и личных кодов
   - Лимит использований
   - Скидка в процентах

5. **admin_users** - Администраторы системы
   - Поле is_super_admin для разделения прав
   - Супер-админ может управлять сообщениями и администраторами

6. **bot_messages** - Управляемые сообщения
   - WELCOME - приветствие при старте
   - PAYMENT_DETAILS - реквизиты оплаты
   - PAYMENT_CONFIRMED - подтверждение оплаты
   - SURVEY_REJECTED - отклонение анкеты

## 🔄 Поток взаимодействия пользователя

```
Пользователь /start
    ↓
Показ приветствия (из BotMessage)
    ↓
Заполнение анкеты (вопросы из survey.json)
    ↓
Сохранение в БД (survey_submissions) со статусом PENDING_REVIEW
    ↓
ОЖИДАНИЕ АДМИНА
    ↓
Админ принимает решение (approve/reject)
    ↓
Если отклонено → REJECTED + уведомление
Если одобрено → APPROVED + отправка реквизитов
    ↓
ОЖИДАНИЕ ОПЛАТЫ (PENDING_PAYMENT)
    ↓
Администратор подтверждает оплату
    ↓
Создание Subscription + статус PAID
    ↓
✅ Доступ в клуб
```

## 👨‍💼 Панель администратора

### Обычный Администратор может:
1. **Модерация анкет**
   - Просмотр анкет на рассмотрение
   - Одобрение с возможностью выставления персональной скидки
   - Отклонение с комментарием

2. **Подтверждение платежей**
   - Просмотр анкет в статусе PENDING_PAYMENT
   - Подтверждение оплаты → создание подписки

3. **Управление анкетой (survey.json)**
   - Добавление вопросов
   - Редактирование вопросов
   - Удаление вопросов
   - Поддержка типов: text, yes_no, multiple_choice

### Супер-администратор дополнительно может:
1. **Управление сообщениями**
   - Редактирование приветствия
   - Редактирование реквизитов оплаты
   - Редактирование сообщения подтверждения оплаты
   - Редактирование сообщения об отклонении

2. **Управление администраторами**
   - Добавление администраторов
   - Удаление администраторов
   - Список всех администраторов

3. **Управление тарифными планами**
   - Создание новых планов
   - Удаление планов
   - Динамическое изменение предложений

4. **Управление промокодами** (в разработке)
   - Создание групповых кодов
   - Создание личных кодов
   - Управление лимитами использований
