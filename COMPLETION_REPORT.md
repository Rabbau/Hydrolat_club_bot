## 🔄 Процесс работы системы

### 1. Запуск бота
```
python main.py

📊 Applying database migrations...
✅ Database migrations applied
💬 Initializing bot messages...
✅ Bot messages initialized
🤖 Bot is starting polling...
```

### 2. Пользовательский поток
```
Пользователь /start
    ↓
👋 Приветствие (из БД)
    ↓
📋 Заполнить анкету
    ↓
Ответы на вопросы (1-N)
    ↓
✅ Анкета отправлена → PENDING_REVIEW
    ↓
⏳ Ожидание админа (проверка статуса возможна)
    ↓
Админ рассматривает:
  → ОДОБРЕНО: → PENDING_PAYMENT + реквизиты
  → ОТКЛОНЕНО: → REJECTED + комментарий
    ↓
(Если одобрено) Оплата по реквизитам
    ↓
Админ подтверждает оплату → PAID
    ↓
🎉 Доступ в клуб! Подписка активна
```

### 3. Админ-панель
```
Администратор /start
    ↓
🔐 Панель администратора
    ├── 📋 Анкета
    ├── 👥 Модерация
    │   ├── 📋 Проверить анкеты (PENDING_REVIEW)
    │   └── 💳 Подтвердить платежи (PENDING_PAYMENT)
    └── 💾 История решений

Супер-администратор /start
    ↓
🔐 Панель администратора (расширенная)
    ├── 👥 Модерация (всё из обычного админа)
    ├── 🎟️ Промокоды
    ├── 👤 Администраторы
    │   ├── ➕ Добавить
    │   ├── ➖ Удалить
    │   └── 👥 Список
    ├── 💰 Тарифные планы
    │   ├── ➕ Создать
    │   └── ➖ Удалить
    └── 💬 Сообщения
        ├── 👋 Приветствие
        ├── 💳 Реквизиты
        ├── ✅ Подтверждение оплаты
        └── ❌ Отклонение
```

## 🎓 Как использовать систему

### Для обычного пользователя
1. Напишите боту `/start`
2. Нажмите "📋 Заполнить анкету"
3. Ответьте на все вопросы
4. Нажмите "📊 Проверить статус" чтобы отслеживать анкету
5. После одобрения получите реквизиты
6. Оплатите по реквизитам
7. Дождитесь подтверждения от администратора

### Для администратора
1. Добавьте себя в БД как администратор
2. Напишите боту `/start`
3. Перейдите в "👥 Модерация"
4. "📋 Проверить анкеты" - просмотрите новые анкеты
5. Одобрите или отклоните
6. "💳 Подтвердить платежи" - когда пользователь оплатит
7. Подтвердите оплату

### Для супер-администратора
1. Добавьте себя с флагом `is_super_admin = true`
2. Используйте все функции администратора плюс:
3. "👤 Управление администраторами" - добавляйте/удаляйте админов
4. "💰 Тарифные планы" - создавайте разные планы оплаты
5. "💬 Сообщения" - редактируйте все сообщения бота
6. "🎟️ Промокоды" - создавайте промокоды для скидок
